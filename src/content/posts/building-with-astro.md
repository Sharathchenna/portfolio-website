# Building a Portfolio with Astro

Astro has quickly become one of my favorite frameworks for building static websites. Its component-based architecture, combined with excellent performance characteristics, makes it an ideal choice for developer portfolios.

## Why Astro?

Astro offers several advantages that made it perfect for my portfolio:

1. **Zero JS by default**: Astro ships zero JavaScript to the browser unless explicitly needed, resulting in lightning-fast page loads.

2. **Use any UI framework**: I could use React components where I needed interactivity while keeping most pages static.

3. **Content Collections**: The built-in content management system makes organizing blog posts and projects a breeze.

4. **Markdown support**: Writing content in Markdown is convenient and maintainable.

## Getting Started

Setting up an Astro project is straightforward:

```bash
# Create a new project with npm
npm create astro@latest
```
