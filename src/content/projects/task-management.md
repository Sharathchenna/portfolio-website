# Task Management App

A sleek, intuitive task management application built to boost productivity. This project features:

- Kanban board interface for task organization
- Real-time sync across devices
- Task categorization and priority setting
- Due date reminders and notifications
- Collaboration features for team projects

Built with Vue.js for the frontend and Firebase for backend services including authentication, real-time database, and cloud functions.
